Nombre del atributo Descripción Ejemplo
background-color Cambia el color de fondo, se puede expresar el color en hexadecimal o por el nombre de color si es uno de los que está predefinido background-color:red
background-color:#fafafa;
width Esta propiedad cambia el tamaño del ancho para un elemento, se puede expresar en porcentaje, pixeles, em, puntos... width:120px;
width:33%;
padding Designa el espacio entre un elemento y su contenido, es decir, es el espacio que dejará respecto al borde del contenedor un elemento interno. Al igual que width y la mayoría de las propiedades CSS se puede expresar en pixeles, porcentaje... padding: 20px
También es posible designar cada lada independientemente:
padding-top:20px;
padding-right:20px;
padding-bottom:20px;
padding-left:20px;
float Permite designar como flotante una caja la cual pasará a ser un elemento de bloque que se adaptará (flotará) respecto a otras cajas mientras haya espacio. Es posible declarar que flote a la derecha (right) o a la izquierda (left) float:left;
display Es el modo de comportamiento para la caja, permitiendo o forzando ser un elemento de bloque (block) o elemento de linea (inline) incluso mezclar ambos (inline-block) display:block;
display: inline;
display:inline-block;
#nombredeseccion Es un identificador único que permite acceder a un elemento previamente designado con este. Se tiene que aplicar en HTML como atributo de la etiqueta id="nombre_del_identificador"
text-decoration Nos deja establecer la decoración del texto como el subrayado, tachado... e incluso no tener decoración (none) text-decoration:none;
text-decoration:underline;
text-decoration:overline;
.nombredeclase Establece un selector de clase aplicable a varios elementos, permite cambiar de una vez varios elementos indistintamente de donde se encuentren en el documento HTML. Se designa como atributo de una etiqueta HTML con el atributo "class" y desde la hoja de estilos CSS lo llamamos con un punto delante del nombre ".nombredeclase" class="nombredeclase"
font-family Cambia la fuente de texto por la elegida font-family:monospace;
border Establece o elimina un borde para el elemento. Se puede designar de una sola vez el tamaño en pixeles, el tipo de borde (solid, groove...) y el color tanto en hexadecimal como por su nombre si es de los reconocidos border: 1px solid black;
border: 3px groove #fafafa;
